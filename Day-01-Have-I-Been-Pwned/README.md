# Day 01 - Have I Been Pwned? (HIBP)

## Objective
Learn what a data breach is, check whether my email has been exposed in known breaches, and understand why cybersecurity awareness is important.

---

## Activity Performed
- Visited Have I Been Pwned (HIBP).
- Checked if my email appeared in public data breaches.
- Read about the breaches affecting my email.
- Researched SOC Analyst and GRC Analyst job postings to understand industry requirements.

---

## What I Learned
- A data breach occurs when personal information is exposed or stolen.
- Password reuse significantly increases security risks.
- Multi-Factor Authentication (MFA) adds an additional layer of protection.
- Monitoring breaches helps users take preventive actions.

---

## Key Takeaways
- Use unique passwords for every account.
- Enable MFA wherever possible.
- Regularly monitor accounts for security incidents.
- Understanding job requirements early helps create a focused learning path.

## Career Research

### SOC Analyst – Common Skills Found in Job Postings
- Networking Fundamentals (TCP/IP, DNS, HTTP)
- Linux and Windows Administration
- SIEM Tools (Splunk, Microsoft Sentinel, QRadar)
- Incident Response
- Log Analysis
- Threat Detection and Monitoring
- Basic Scripting (Python, PowerShell)
- Security Frameworks and Best Practices

### GRC Analyst – Common Skills Found in Job Postings
- Risk Management
- Security Policies and Procedures
- Compliance Standards (ISO 27001, NIST, SOC 2)
- Governance Frameworks
- Risk Assessments
- Documentation and Reporting
- Audit Support
- Communication and Stakeholder Management
