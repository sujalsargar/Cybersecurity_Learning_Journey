## The Four Main Parts of a Computer

### 1. CPU (Central Processing Unit)
- Known as the "brain" of the computer.
- Executes instructions and performs calculations.
- Manages and coordinates the activities of other components.
- Modern processors can handle many processes and applications simultaneously.

### 2. RAM (Random Access Memory)
- Temporary memory used while programs are running.
- Stores data that the CPU needs quickly.
- Data in RAM is lost when the computer is turned off.

### 3. Storage
- Permanent memory used to store files, applications, and the operating system.
- Examples: HDD (Hard Disk Drive) and SSD (Solid State Drive).
- Stores documents, images, videos, and program files.

### 4. Network Interface
- Allows a computer to communicate with other devices and the internet.
- Examples: Wi-Fi adapters and Ethernet cards.
- Enables browsing, downloading, and network communication.

---

## Important Learnings

### Multiple Programs Run at the Same Time
- A computer runs many processes simultaneously.
- Even when only a few applications are open, the operating system may be running hundreds of background processes.

### Every File Has an Identity
Every file has:
- A name
- A location (path)
- An address in the file system

Examples:

Windows:
```text
C:\Users\Sujal\Documents\report.pdf
```

Linux:
```text
/home/sujal/Documents/report.pdf
```

The file path is an important part of a file's identity and helps the operating system locate it.

### Network Security Concept
A simple but powerful idea:

> "No internet connection = No remote theft."

Although physical threats still exist, many cyberattacks require network connectivity. Understanding networks is essential for understanding cybersecurity.

## Introduction to EDR (Endpoint Detection and Response)

I learned the basics of EDR tools.

### What is EDR?
EDR stands for Endpoint Detection and Response.

EDR tools:
- Monitor endpoints such as laptops and desktops.
- Detect suspicious activities.
- Generate security alerts.
- Help investigate and respond to incidents.

### Examples of EDR Tools
- Microsoft Defender for Endpoint
- CrowdStrike Falcon
- SentinelOne
- Sophos Intercept X

### Why EDR Matters
- Detects malware and suspicious behavior.
- Helps security teams investigate incidents.
- Improves visibility into endpoint activities.
- Supports faster incident response.
